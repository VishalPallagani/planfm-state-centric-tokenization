# Statistical Summaries

These files are compact secondary tables generated from completed tokenizer-study CSVs and compact analysis outputs.

| File | Purpose |
| --- | --- |
| `README.md` | This directory guide. |
| `anova_additive.csv` | Additive effect summary over held-out seed-level scores. |
| `anova_interaction.csv` | Interaction-model summary for the factorial result analysis. |
| `representation_stats.csv` | Tokenizer representation dimensions and fit-strategy summary. |
| `baseline_domain_summary.csv` | Domain-resolved classical-planner baseline summary. |
| `baseline_macro_summary.csv` | Macro-level classical-planner baseline summary. |
| `effort_table.csv` | Search-effort diagnostics for the strongest learned configuration. |
| `failure_diagnostics.csv` | Extrapolation failure-mode decomposition. |
| `pairwise_test_significance_by_split.csv` | Held-out pairwise tokenizer tests resolved by interpolation/extrapolation split. |
| `pairwise_test_significance_overall.csv` | Overall held-out pairwise tokenizer tests. |
| `regime_test_significance_overall.csv` | Held-out paired domain-dependent versus all-domain transfer tests. |
| `split_significance_counts.csv` | Counts of Holm-significant tokenizer contrasts by split and method family. |
| `tokenizer_settings.csv` | Tokenizer construction settings and dimensions. |
| `winner_summary.csv` | Winner and runner-up summary for each protocol/predictor/formulation family. |
